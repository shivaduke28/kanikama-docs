# Texture Resizing

You can select **Texture Resize Type** in the Inspector of UnityBakingSettingAsset. 

<a href="https://gyazo.com/dc48ae573b059bb0c301262f80e71c75"><img src="https://i.gyazo.com/dc48ae573b059bb0c301262f80e71c75.png" alt="Image from Gyazo" width="728"/></a>

This value is used for lightmap arrays, directional lightmap arrays, and LTC visibility maps. Texture resizing is performed on the asset creation step; simply change the Resize Type and re-run the asset creation; no baking is required.


The left is ResizeType One and the right is ResizeType One Half (asset size is 1/4).
